# Support Utilisateur et Instructions de Création d'Issues

## Support Utilisateur
Pour toute demande de support, veuillez contacter notre équipe d'assistance à support@botaniq.com. Nous sommes là pour vous aider avec toutes les questions ou problèmes que vous pourriez rencontrer en utilisant Botaniq.

## Création d'Issues
Si vous rencontrez un bug ou souhaitez demander une nouvelle fonctionnalité, veuillez créer une issue dans notre référentiel GitHub. Suivez les étapes ci-dessous :

1. Allez à l'[onglet issues](https://github.com/jessy2027/Botaniq-support/issues) dans notre référentiel.
2. Cliquez sur le bouton "New Issue" (Nouvelle Issue).
3. Fournissez un titre et une description clairs pour l'issue, en incluant les étapes de reproduction si applicable.
4. Étiquetez l'issue de manière appropriée (bug, feature request, question, etc.).
5. Soumettez l'issue !

Merci pour vos contributions et de nous aider à améliorer Botaniq !
