# User Support and Issue Creation Instructions

## User Support
For any support-related inquiries, please reach out to our support team at support@botaniq.com. We are here to assist you with any questions or issues you may encounter while using Botaniq.

## Issue Creation
If you encounter a bug or wish to request a feature, please create an issue in our GitHub repository. Follow the steps below:

1. Go to the [issues tab](https://github.com/jessy2027/Botaniq-support/issues) in our repository.
2. Click on the "New Issue" button.
3. Provide a clear title and description for the issue, including steps to reproduce if applicable.
4. Label the issue appropriately (bug, feature request, question, etc.).
5. Submit the issue!

Thank you for your contributions and for helping us improve Botaniq!
